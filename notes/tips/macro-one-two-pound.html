井号(#)和双井号(##)在C语言中的作用

  • 在C的宏(macro)语言中,是可以用#作为字符化的工具, 而##作为连接字符的工具, 下面 一个例子就说明概况了它的用法 
    #include <stdio.h>
#define PRINT(NAME) printf("token"#NAME"=%d\n", token##NAME)
int main(int argc, char *argv[])
{
    int token9=9;
    PRINT(9);
    return 0;
}
/* --------output is following----------------------*/
/* token9=9                                         */
